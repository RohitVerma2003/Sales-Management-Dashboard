# 📊 Sales Management Dashboard

A Power BI dashboard built to analyze sales performance, profitability, customer purchasing patterns, and promotional campaign effectiveness using a retail sales dataset.

## 🚀 Project Overview

This project focuses on transforming raw sales data into meaningful business insights through data cleaning, analysis, and interactive visualizations. The dashboard helps identify top-performing products, sales trends, profitable regions, and the impact of promotional campaigns.

## 🛠️ Tools & Technologies

* Power BI
* Microsoft Excel
* Data Cleaning & Transformation
* Data Modeling
* DAX Measures

## 📈 Dashboard Features

* Sales trend analysis over time
* Sales distribution by city
* Profit vs Net Sales analysis
* Promotion-wise discount analysis
* Top & Bottom 5 products by:

  * Sales
  * Units Sold
  * Profit
* Date comparison dashboard
* Interactive filters for:

  * Date
  * Customer
  * Product
  * Promotion
* Detailed sales transaction table
